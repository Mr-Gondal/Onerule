# Contributing to OneRule

Thanks for helping out! The most valuable contributions are **new targets** (support for another AI coding agent).

## Setup

```bash
npm install
npm run dev -- sync   # run the CLI from source
npm test              # run the test suite
npm run build         # produce dist/
```

## Adding a new target

1. Open [`src/targets.ts`](src/targets.ts).
2. Add one entry to the `TARGETS` registry:

   ```ts
   myagent: {
     id: "myagent",
     label: "My Agent",
     outputPath: ".myagent/rules.md",
     render: (body) => `<!-- generated by onerule -->\n\n${body}\n`,
   },
   ```

3. Add `"myagent"` to the defaults in [`src/config.ts`](src/config.ts) if it should be on by default.
4. Add a row to the table in `README.md`.
5. Run `npm test`.

That's it — most target PRs are under 15 lines.

## Guidelines

- Keep dependencies minimal and the CLI offline/local-first.
- Every command takes an optional `cwd` so it stays unit-testable.
- Add or update a test in `tests/` for behaviour changes.
- Be kind in issues and reviews.
